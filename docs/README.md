# Documentación de Biux

Esta carpeta contiene toda la documentación del proyecto Biux.

## Estructura

### `/firebase`
Documentación relacionada con la configuración e implementación de Firebase:
- `ENABLE_FIREBASE_PHONE_AUTH_MANUAL.md` - Manual para habilitar autenticación por teléfono
- `FIREBASE_PHONE_AUTH_IMPLEMENTATION.md` - Implementación de autenticación por teléfono
- `FIREBASE_PHONE_AUTH_SETUP.md` - Configuración inicial de autenticación por teléfono

### `/implementation`
Documentación de implementación de features específicas:
- `INSTAGRAM_LAYOUT_IMPLEMENTATION.md` - Implementación de layout estilo Instagram
- `STORIES_IMPLEMENTATION_SUMMARY.md` - Resumen de implementación de historias

## Otros recursos
- Archivos HTML de ejemplo y simuladores: Ver `/web/examples/`
- Scripts de verificación y seed: Ver `/scripts/`
- Código de debug: Ver `/lib/core/debug/`
- Ejemplos de código: Ver `/lib/core/examples/`
