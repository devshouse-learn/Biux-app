# Guías de Implementación

Esta carpeta contiene documentación detallada sobre la implementación de features específicos en la aplicación Biux.

## Documentos Disponibles

### Layout y UI
- **INSTAGRAM_LAYOUT_IMPLEMENTATION.md** - Guía de implementación del layout estilo Instagram para el feed social

### Features Sociales
- **STORIES_IMPLEMENTATION_SUMMARY.md** - Resumen de la implementación de historias (stories) al estilo Instagram/WhatsApp

## Propósito

Estos documentos sirven como:
- Referencia para futuros desarrollos similares
- Documentación de decisiones de diseño y arquitectura
- Guías paso a paso para implementar features complejos
- Registro del proceso de desarrollo de features clave

## Convenciones

Cada documento de implementación debería incluir:
- Descripción del feature
- Arquitectura y estructura de archivos
- Dependencias necesarias
- Pasos de implementación
- Problemas conocidos y soluciones
- Capturas de pantalla o diagramas cuando sea relevante
