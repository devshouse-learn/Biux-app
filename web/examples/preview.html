📱 Características de la App

  • 📖
    Historias y Feed Social
    Comparte experiencias, foto y videos de tus rodadas
  • 🚴
    Gestión de Rodadas
    Crea, únete y gestiona rodadas con otros ciclistas
  • 👥
    Comunidades
    Forma o únete a grupos de ciclismo locales
  • 🗺️
    Mapa Integrado
    Google Maps para visualizar rutas y puntos de encuentro
  • 🔐
    Autenticación Firebase
    Registro seguro con Google, Apple y correo
  • 🛍️
    Tienda
    Compra y vende accesorios para bicicletas
  • 🔔
    Notificaciones Push
    Recibe alertas en tiempo real de eventos importantes
  • 🚲
    Registro de Bicicletas
    Gestiona tus bicicletas y reporta robos

🚀 Próximos Pasos

  1. Instala Flutter: Descárgalo de https://flutter.dev/docs/get-started/install
  2. Ejecuta `flutter pub get`: Para instalar todas las dependencias
  3. Inicia el emulador: Android Studio o iOS Simulator
  4. Ejecuta `flutter run`: Desde VS Code o terminal
  5. Abre en depurador: Presiona F5 o usa el Run & Debug
