# Ejemplos Web de Biux

Esta carpeta contiene archivos HTML de ejemplo y herramientas de simulación para el proyecto Biux.

## Archivos

- `preview.html` - Vista previa de componentes o funcionalidades
- `simulador.html` - Simulador para pruebas de funcionalidades

## Uso

Estos archivos pueden abrirse directamente en un navegador para previsualizar o simular funcionalidades de la aplicación Biux.
